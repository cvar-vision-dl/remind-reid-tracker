Abstract

REMIND is an online multi-object re-identification tracker designed to keep object identities consistent across long indoor sequences. The method combines DINO-based appearance descriptors, semantic part descriptors, local background context, relational neighbor memory, and cautious memory updates.

Instead of relying only on frame-to-frame position, REMIND builds persistent object memory and uses it to reconnect objects after occlusions, viewpoint changes, illumination changes, and re-appearances.

Main Results

REMIND explicitly models ambiguous and uncertain cases. For quantitative evaluation, ambiguous/provisional labels are collapsed into a final identity decision so standard tracking metrics can be computed. This collapsed output is used only for reporting: internally, the method does not always force an immediate identity label.
