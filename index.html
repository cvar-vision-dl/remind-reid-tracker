Abstract

REMIND is an online multi-object re-identification tracker designed to keep object identities consistent across long indoor sequences. The method combines DINO-based appearance descriptors, semantic part descriptors, local background context, relational neighbor memory, and cautious memory updates.

Instead of relying only on frame-to-frame position, REMIND builds persistent object memory and uses it to reconnect objects after occlusions, viewpoint changes, and re-appearances.

Method Overview

1. Perception Segment current objects and extract DINO global, part-level, and background descriptors.
2. Association Compare current detections against tracked identities and solve global assignment with Hungarian matching.
3. Context Use reliable anchors, neighbor sets, and distance memory to disambiguate difficult cases.
4. Update Update object memory only when evidence is reliable, while keeping ambiguous and provisional states explicit.
